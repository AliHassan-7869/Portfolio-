# 🚀 Ali Hassan — AI Engineer Portfolio

A high-end, immersive developer portfolio showcasing AI engineering and generative AI work — multi-agent systems, LLM applications, and deployed AI backends.

## ✨ Features

- **Premium Case Studies**: Immersive detail pages for each AI project, covering architecture, tech stack, and results.
- **Dynamic Animations**: Powered by **GSAP** (GreenSock Animation Platform) for smooth transitions and interactive elements.
- **Inertia Scrolling**: Integrated with **Lenis** for a refined, agency-level scrolling experience.
- **Modern UI/UX**: Clean, professional dark-mode design aesthetic.
- **Responsive Design**: Built to look stunning on all devices and screen sizes.

## 🛠️ Tech Stack

- **Core**: HTML5, Vanilla CSS, JavaScript (ES6+)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Animations**: [GSAP](https://greensock.com/gsap/)
- **Smooth Scroll**: [Lenis](https://lenis.darkroom.engineering/)
- **Icons**: [Lucide](https://lucide.dev/)

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production

```bash
npm run build
```

## 📂 Project Structure

- `index.html`: Main landing page.
- `detail.html`: AI project case-study page template.
- `main.js`: Core logic and animations for the landing page.
- `detail.js`: Logic for the case-study detail pages.
- `data.js`: Project and skills data (edit this to update project content).
- `style.css`: Main stylesheet containing the design system.

## ✏️ Customizing

- **Projects & Skills**: Edit `data.js` — each entry has `title`, `tagline`, `icon`, `description`, `techStack`, `features`, `workflow`, `challenge`, `solution`, and `results`.
- **Profile photo**: Replace `/public/profile.jpeg` with your own photo.
- **Contact info & links**: Update the contact section in `index.html` and the Github/LinkedIn/Huggingface links.
